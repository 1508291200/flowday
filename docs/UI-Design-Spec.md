# FlowDay 界面设计规范

**版本**: V1.0  
**日期**: 2026-04-29  
**状态**: 正式发布

---

## 1. 设计原则

| 原则 | 说明 |
|-----|------|
| **简洁克制** | 减少视觉噪音，聚焦内容本身 |
| **一致性** | 相同功能使用相同视觉表达 |
| **层级清晰** | 通过颜色、大小、间距表达信息层级 |
| **即时反馈** | 用户操作后 100ms 内给予视觉反馈 |

---

## 2. 色彩系统

### 2.1 主色板

```
Primary Blue:    #3B82F6  → 主操作、选中状态、链接
Background:      #F9FAFB  → 页面背景
Surface:         #FFFFFF  → 卡片、节点背景
Border:          #E5E7EB  → 分割线、边框
```

### 2.2 语义色

| 用途 | 颜色 | HEX |
|-----|------|-----|
| 成功/完成 | Green | `#10B981` |
| 警告/即将过期 | Orange | `#F59E0B` |
| 错误/已过期 | Red | `#EF4444` |
| 信息 | Blue | `#3B82F6` |
| 禁用 | Gray | `#9CA3AF` |

### 2.3 重要度色系

| 级别 | 名称 | HEX | 使用场景 |
|-----|------|-----|---------|
| 1 | 最低 | `#9E9E9E` | 可延后的任务 |
| 2 | 低 | `#4CAF50` | 常规任务 |
| 3 | 中 | `#FFC107` | 需关注 |
| 4 | 高 | `#FF9800` | 重要任务 |
| 5 | 最高 | `#F44336` | 紧急任务 |

**视觉表达**: 左侧 4px 竖条 + 8×8 圆点指示器

## 3. 字体系统

### 3.1 字体栈

```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

### 3.2 字号规范

| 名称 | 尺寸 | 行高 | 用途 |
|-----|------|------|-----|
| xs | 12px | 16px | 标签、辅助信息 |
| sm | 14px | 20px | 节点标题、按钮 |
| base | 16px | 24px | 正文内容 |
| lg | 18px | 28px | 页面标题 |
| xl | 20px | 28px | 主标题 |

### 3.3 字重

| 字重 | 值 | 用途 |
|-----|----|----|
| Normal | 400 | 正文、描述 |
| Medium | 500 | 标题、标签 |
| Semibold | 600 | 强调标题 |

## 4. 间距系统

采用 **8px 基准网格**：

```
4px  → 紧凑元素内间距
8px  → 元素内间距、小组件间距
12px → 中等间距
16px → 卡片内间距、组件间距
24px → 区域间距
32px → 页面边距
```

| lg | 8px | 卡片、节点 |

## 7. 组件规范

### 7.1 按钮 Button

```
高度: sm=28px | md=32px | lg=40px
内边距: sm=8px 12px | md=12px 16px | lg=16px 24px
圆角: 6px
过渡: 150ms ease
```

**状态样式**:

| 变体 | 默认 | 悬浮 | 按下 |
|-----|------|------|------|
| Primary | bg-blue-500 | bg-blue-600 | bg-blue-700 |
| Secondary | bg-gray-100 border | bg-gray-200 | bg-gray-300 |
| Ghost | transparent | bg-gray-100 | bg-gray-200 |
| Danger | bg-red-500 | bg-red-600 | bg-red-700 |

### 7.2 输入框 Input

```
高度: 36px
内边距: 12px
边框: 1px solid #D1D5DB
圆角: 6px
聚焦: ring-2 ring-blue-400 border-blue-400
```

### 7.3 卡片/节点 Card

```
内边距: 12px 16px
边框: 2px solid #E5E7EB
圆角: 8px
背景: #FFFFFF
阴影: shadow-sm → shadow-md (hover)
```

## 8. 视图设计

### 8.1 列表视图 ListView

**布局结构**:
```
┌─────────────────────────────────────┐
│ Header (h-14)                       │
├─────────────────────────────────────┤
│ Toolbar (h-10)  [按钮组]            │
├─────────────────────────────────────┤
│                                     │
│  ▸ 我的日程                         │
│    ✓ ├─ 任务A           [操作按钮]  │
│      ├─ 任务B           [操作按钮]  │
│        └─ 子任务B1      [操作按钮]  │
│                                     │
│  (虚拟滚动区域)                     │
│                                     │
├─────────────────────────────────────┤
│ StatusBar (h-8)  节点: 12 | 完成: 5 │
└─────────────────────────────────────┘
```

**行样式**:
```css
height: 40px;
padding-left: baseIndent + (depth × 24px);
hover: background-gray-50;
selected: background-blue-50;
```

**层级指示器**:
- 树形连接线: 1px solid #D1D5DB
- 折叠箭头: 16×16, 灰色, 90° 旋转动画

### 8.2 思维导图视图 MindMapView

**布局结构**:
```
┌─────────────────────────────────────┐
│ Header                              │
├─────────────────────────────────────┤
│ Toolbar  [添加][适应][+/-][撤销]    │
├─────────────────────────────────────┤
│                        ┌──────────┐ │
│                        │ MiniMap  │ │
│  ┌────────┐            │ (右下角) │ │
│  │我的日程│──┬──▶ ┌────┴────┐    │ │
│  └────────┘  │    │ 任务A   │    │ │
│              ├──▶ ├─────────┤    │ │
│              │    │ 任务B   │    │ │
│              │    └─────────┘    │ │
│              └──▶ ┌─────────┐    │ │
│                   │ 子任务  │    │ │
│                   └─────────┘    │ │
│  (可缩放/平移画布)                 │
├─────────────────────────────────────┤
│ StatusBar                           │
└─────────────────────────────────────┘
```

**节点布局参数**:
```javascript
{
  nodeWidth: 180,      // 节点宽度
  nodeHeight: 40,      // 节点高度

  verticalGap: 20,     // 垂直间距（兄弟）

```

**连接线样式**:
```css
type: 'step';
stroke: #D1D5DB;
strokeWidth: 2;
```

**悬浮操作面板**:
```
┌───────────────────┐
│  + │ 编辑 │ 删除  │
└───────────────────┘
position: absolute; top: -32px; left: 50%; transform: translateX(-50%);
background: white; shadow: md; radius: 4px; padding: 4px 8px;
```

## 9. 交互规范

### 9.1 状态反馈

| 操作 | 反馈 | 时长 |
|-----|------|-----|
| 悬浮节点 | shadow-sm → shadow-md | 150ms |
| 点击选中 | border-gray → border-blue-500 ring | 100ms |
| 添加节点 | 淡入动画 | 200ms |
| 删除节点 | 淡出动画 | 150ms |
| 保存编辑 | 边框闪烁蓝色 | 100ms |

### 9.2 过渡动画

```css
/* 通用过渡 */
transition: all 150ms ease-out;

/* 悬浮面板 */
transition: opacity 150ms ease-out, transform 150ms ease-out;

/* 折叠展开 */
transition: transform 200ms ease-out;
```

### 9.3 键盘支持

| 按键 | 功能 |
|-----|------|
| Enter | 确认编辑 |
| Escape | 取消编辑/关闭弹窗 |
| Delete | 删除选中节点 |
| Ctrl+Z | 撤销 |
| Ctrl+Shift+Z | 重做 |

## 11. 图标规范

使用 **Heroicons** (outline style)

| 场景 | 图标 |
|-----|------|
| 添加 | `plus` |
| 删除 | `trash` |
| 编辑 | `pencil` |
| 折叠 | `chevron-right` (rotate-90) |
| 展开 | `chevron-right` |
| 撤销 | `arrow-uturn-left` |
| 重做 | `arrow-uturn-right` |
| 放大 | `magnifying-glass-plus` |
| 缩小 | `magnifying-glass-minus` |
| 适应 | `arrows-pointing-in` |

**尺寸**:
- 按钮内图标: 16×16 (w-4 h-4)
- 工具栏图标: 20×20 (w-5 h-5)

## 12. 无障碍规范

| 要求 | 实现 |
|-----|------|
| 焦点可见 | `focus:ring-2 focus:ring-blue-400` |
| 对比度 | 文字与背景 ≥ 4.5:1 |
| 焦点顺序 | Tab 顺序与视觉顺序一致 |
| ARIA标签 | 按钮、输入框添加 aria-label |
